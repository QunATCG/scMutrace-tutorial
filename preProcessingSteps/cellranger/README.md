# Example codes for Cell Ranger

https://www.10xgenomics.com/support/software/cell-ranger/latest

> All analyses were performed on HPC clusters provided by the PDC Center for High Performance Computing.

https://www.pdc.kth.se/

https://support.pdc.kth.se/doc/

https://slurm.schedmd.com/documentation.html


- cell_ARC.slurm (Cell Ranger ARC's pipelines analyze sequencing data produced from Chromium Single Cell Multiome ATAC + Gene Expression. https://www.10xgenomics.com/support/software/cell-ranger-arc/latest)

- cell_ATAC.slurm (Cell Ranger ATAC's pipelines analyze sequencing data produced from Chromium Epi ATAC libraries. https://www.10xgenomics.com/support/software/cell-ranger-atac/latest)

- cell_RNA.slurm (Cell Ranger provides a set of analysis pipelines designed to process 3' Chromium Single Cell Gene Expression, 5' Chromium Single Cell Immune Profiling, and Flex datasets. https://www.10xgenomics.com/support/software/cell-ranger/latest)
